# go\_console\_chat  

突貫アプリケーション of the year

---
![demo](https://raw.githubusercontent.com/ucpr/go_console_chat/master/img/demo.gif "demo")


## 使い方
wsサーバーの起動
```
$ go run main.go
```

TUIアプリケーションの起動
```
$ go run cmd/chat_tui_client/main.go
```
